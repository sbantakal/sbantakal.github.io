Work Samples

GAN vs VAE: A Comparative Analysis

My first foray into the field of machine learning; an attempt to understand supervised and unsupervised learning in artificial neural networks.
The analysis is based on the observations made from training the Generative Adversarial Network (GAN) & Variational Auto Encoder (VAE) on MNIST handwriting data.

P2P File Transfer

A file transfer simulator, built on notions similar to those employed by the prevalent Bit Torrent application.
The application leverages Java's socket programming & mutithreading capabilities to implement its various operation.

Twitter Simulator

To explore distributed communications, we built a web application using Elixir Pheonix Framework.
The simulator creates user accounts with random usernames, assigns subscribers to each user using Zipf distribution.
Key twitter functions such as tweeting, re-tweeting, follow, unfollow were implemented.
Youtube demo of the project can be found here.

Crime Data Visulaization Tool

The motive of the project was to develop an real time web application that could scour through millions of criminal records and display key trends, patterns in a visually appealing manner.
Criminal records made available to the public by the Chicago Police Department were used as dataset for the project.
The web application was built using Python Django Framework, Chart.js library was used to display data as graphs & heatmaps for logical visualization.
